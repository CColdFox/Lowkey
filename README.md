# Lowkey
(De)crypts a file in SHA256.
## How to install?
1. Download this repository by pressing on the green `Code` button at the top right corner of the page -> `Download ZIP`
3. Extract it where you want, then open the extracted folder.
4. Run the `install-dependencies.bat` file and wait for it to close automatically, then run `main.py`
5. You're all set!
